# 再入院预测

## 任务描述

在给定的数据集上采用合适的数据挖掘/机器学习算法，预测病人是否会在短时间内再入院。该任务可以看成是一个多分类问题，根据数据特征，将数据分成三类：

"NO"、"<30"、" >30"。

## 数据集描述

**Diabetes 130-US hospitals for years 1999-2008 Data Set**

该数据集中包含了1999年~2008年这10年期间美国130家医院等的临床信息，其中这些数据都满足以下条件：

- 都是住院病人
- 都是糖尿病患者
- 逗留时间至少为1天，最多14天
- 在期间进行了相应的测试以及药物治疗。

数据中包含了例如种族，性别之类的等属性。
